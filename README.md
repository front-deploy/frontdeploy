# Frontdeploy

Frontdeploy is a simple Chrome Extension designed for scanning X (Twitter) narratives and trends.

## Structure
- `frontdeploy-project-source/`: The Chrome Extension source code.
- `frontdeploy-web/`: Landing page website.

## Setup
1. Open `chrome://extensions`
2. Enable Developer Mode
3. Load unpacked extension from `frontdeploy-project-source/apps/extension/build/chrome-mv3-prod`
