# Code of Conduct

## Our Pledge

We aim to keep this project practical, respectful, and safe for contributors
and users.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code, design, safety, and user impact.
- Assume good intent while still reviewing carefully.
- Report security-sensitive issues privately.

## Unacceptable Behavior

- Harassment, abuse, or personal attacks.
- Publishing private information or secrets.
- Encouraging unsafe wallet, key, or trading behavior.
- Submitting code that intentionally misleads users.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors
that violate these expectations.
